#### 介绍
墙贴

#### 软件架构
基于springboot spring data jpa htm+ajax 实现小贴贴功能

#### 功能介绍
1. 发布贴贴内容
2. 贴贴显示方式，随机弹窗层，玻璃窗形式停靠
3. 排序，点赞，点踩，时间，随机等方式
4. 评论功能实现每个小帖帖内的评论
5. 屏蔽当你选择屏蔽后，该贴贴将不再显示
6. 根据ip地址显示贴贴发布的城市
